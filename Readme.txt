Overview

This project is a web scraper that extracts product details such as titles, prices, and descriptions from Amazon using BeautifulSoup and Pandas. The scraped data is stored in a structured Pandas DataFrame for further analysis or storage.

Features

Extract product titles from Amazon product listings.

Extracts prices, ensuring alignment with titles.

Extracts descriptions, handling missing values gracefully.

Ensures data consistency by aligning all lists.

Converts the extracted data into a Pandas DataFrame for easy manipulation and export.

Requirements

Python 3.x

BeautifulSoup (bs4)

Pandas
